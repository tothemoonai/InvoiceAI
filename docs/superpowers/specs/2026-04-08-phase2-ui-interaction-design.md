# Phase 2 UI 改进设计：响应式布局 + 视觉反馈

> **日期**: 2026-04-08  
> **阶段**: Phase 2 (交互增强)  
> **优先级**: P1  
> **预估工作量**: 4-5 小时

---

## 1. 概述

### 1.1 目标

为 MainPage 添加响应式布局断点切换，并为所有交互组件添加视觉反馈动画。

### 1.2 范围

| 包含 | 不包含 |
|------|--------|
| 三栏断点布局 (>1200 / 900-1200 / <900) | 图片预览增强 (Phase 3) |
| 分类面板可切换显示/隐藏 | 弹出对话框动画 (Phase 3) |
| 按钮悬停/按下动画 | 骨架屏加载 (Phase 3) |
| 列表项选中/悬停视觉反馈 | 矢量图标替换 (Phase 3) |
| 卡片阴影效果 | |

---

## 2. 响应式布局

### 2.1 断点定义

| 断点 | 窗口宽度 | 布局 |
|------|----------|------|
| Expanded | > 1200px | 分类 180px \| 列表 320px \| 详情 Star |
| Standard | 900-1200px | 分类 150px \| 列表 250px \| 详情 Star |
| Compact | < 900px | 列表 Star \| 详情隐藏 (详情改为弹窗) |

### 2.2 实现方案

监听 `OnSizeAllocated`，根据宽度切换 Grid 的 `ColumnDefinitions` 和可见性。

Compact 模式下添加汉堡菜单按钮显示/隐藏分类面板。

---

## 3. 视觉反馈

### 3.1 按钮交互

使用 `VisualStateManager` 添加:
- Normal: 默认状态
- PointerOver: 背景变亮 10%，Scale 1.05
- Pressed: 背景变暗 10%，Scale 0.95

### 3.2 列表项交互

- Normal: 白色背景，浅灰边框，无阴影
- PointerOver: 悬停背景，边框加深，添加小阴影
- Selected: 选中背景色，品牌色边框，中等阴影

### 3.3 加载动画

导入/导出按钮在操作进行中显示 Loading 状态（ActivityIndicator + 文字）。
